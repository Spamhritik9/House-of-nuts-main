## 🔧 STEP 2: CREATE GitHub REPOSITORY

### Go to GitHub.com

1. **Visit** https://github.com/new
2. **Enter repository name:** `house-of-nuts`
3. **Description:** `Premium Makhana E-commerce`
4. **Choose:** Public
5. **Click:** "Create repository"

You'll see a page with these options. Go to Step 3.

---

## 📤 STEP 3: UPLOAD FILES TO GITHUB

### Option A: EASIEST - Upload via GitHub Website (2 minutes)

1. **Go to your GitHub repository** you just created
2. **Click:** "Add file" → "Upload files" button
3. **Drag and drop** the entire `house-of-nuts-complete` folder
   - Or click "select files" and select all files
4. **Scroll down** and click "Commit changes"

✅ **All your files are now on GitHub!**

---

### Option B: Using GitHub Website - Upload Each Folder

If drag & drop doesn't work:

1. **Click** "Add file" → "Upload files"
2. **First**, upload all files from the root folder:
   - package.json
   - next.config.js
   - tailwind.config.js
   - tsconfig.json
   - postcss.config.js
   - data.ts
   - store.ts
   - .gitignore
3. **Commit** with message: "Add root files"
4. **Create folder** `app` → Upload all files from app folder
5. **Create folder** `styles` → Upload globals.css

✅ **Done with GitHub upload!**

---

## 🚀 STEP 4: DEPLOY TO VERCEL

1. **Go to** https://vercel.com/new
2. **Sign up** (if not already)
3. **Click** "Continue with GitHub"
4. **Select your** `house-of-nuts` repository
5. **Click** "Import"
6. **Click** "Deploy"
7. **Wait 3-5 minutes**

✅ **Your website is LIVE!**

You'll get a URL like: `house-of-nuts-xxxxx.vercel.app`

---

## 🌐 STEP 5: ADD YOUR CUSTOM DOMAIN

1. **Buy domain** at Namecheap, GoDaddy, or Google Domains
   - Example: `houseofnuts.com`

2. **In Vercel Dashboard:**
   - Click Settings → Domains
   - Add your domain
   - Copy nameservers

3. **At domain registrar:**
   - Find "Nameservers" settings
   - Replace with Vercel's nameservers
   - Save

4. **Wait 24-48 hours** for DNS to update

✅ **Your site is now at https://houseofnuts.com**

## ❓ HAVING TROUBLE?

### Uploading to GitHub

**Problem:** Files won't upload
**Solution:** 
- Use Chrome browser
- Try uploading folder zip file directly
- Or upload files one by one

### Vercel Deployment

**Problem:** Build fails
**Solution:**
- Wait 2-3 minutes after uploading to GitHub
- Refresh Vercel page
- Click "Redeploy" button

### Custom Domain

**Problem:** Domain not working
**Solution:**
- Wait 24-48 hours (DNS propagation)
- Check nameservers at registrar
- Verify on Vercel that domain is added
